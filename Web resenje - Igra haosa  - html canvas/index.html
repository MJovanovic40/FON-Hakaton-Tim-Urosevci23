Sjerpinski trougao

"Igra Haosa" je algoritam koji je originalno opisan od strane Barnslija 1988. godine.
Algoritam bira tačku nasumično unutar oblasti konveksnog n-ugaonika, potom crta sledeću tačku udaljenu od jednog od nasumično izabrabranih temena n-ugaonika udaljenu za razlomak r cele distance.
Proces se nastavlja iterativno (nakon podešavanja prvobine tačke).
Rezultat ove "igre haosa" je ponekad fraktal.
Trougao Sjerpinskog će biti nacrtan ako se algoritam "igre haosa" konfiguriše za (3, 1/2) - Odnosno trougao je, i razlomak je 1/2.
